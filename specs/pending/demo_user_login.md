# 功能规范 - 用户登录功能

## 项目背景
为移动端应用添加用户登录功能，支持手机号+密码登录和第三方 OAuth 登录。

## 功能描述

### 1. 手机号+密码登录
- 用户输入手机号和密码进行登录
- 手机号格式验证（中国大陆11位）
- 密码长度 8-20 位，包含字母和数字
- 登录成功后返回 JWT Token
- 登录失败显示错误提示

### 2. 验证码登录
- 用户输入手机号，请求短信验证码
- 验证码 6 位数字，有效期 5 分钟
- 每个手机号每分钟最多请求 1 次验证码
- 验证码登录成功后返回 JWT Token

### 3. 第三方登录
- 支持微信登录
- 支持支付宝登录
- 首次第三方登录自动创建账号
- 后续第三方登录绑定已有账号

### 4. 密码管理
- 忘记密码：通过短信验证码重置密码
- 修改密码：需要验证旧密码

## 非功能需求
- 登录接口响应时间 < 500ms（P99）
- 支持并发 1000 QPS
- JWT Token 有效期 7 天
- 登录失败 5 次后锁定账号 30 分钟
- 验证码接口防刷：同一 IP 每小时最多 10 次

## 约束
- 最低支持 Android 8.0 (API 26)
- 网络异常时显示友好提示
- 登录页面需支持深色模式
